# Instructions Git

## Règles de contribution

1. **Restriction `main`** : Il est interdit de push directement sur la branche `main`. Veuillez passer par des Pull Requests.

2. **Nommage des branches** : Utilisez la structure suivante pour vos branches :
   `DX-Nom` ou `DX-Pseudo`
   
   Où `DX` correspond au numéro du livrable (ex: D1, D2, etc.).

   Exemples :
   - `D1-Dupont`
   - `D3-Pseudo`
